# Jackpots-bot-Original
A basic bot that can help you decide between value betting and jackpot betting or both
Its features include:
It uses a Kelly criterion & there is an option to add a Dixon Coles logic if needed. The Dixon Coles usually helps in predicting or adjusting gamestate according to the scoreline.
Intergration with external API key providers to get data that the program will use in making a decision (I've used ODDS football API & API football) feel free to adjust if need be. 
Intergration with Telegram in order for you to receive telegram alerts.
